03 — von Mises–Fisher Loss with Learnable Per-Modality κ (primary framing)

Each concept is modeled as a vMF distribution with shared mean direction μ_c and a learnable per-modality concentration κ_m (paper.md §2.3). Predictions this framing makes:

  • κ_sketch < κ_photo — sketch abstraction variance falls out as low concentration;
  • κ acts as a learned uncertainty weight (gradients scale with κ_m), replacing hand-tuned pairwise loss weights.

Training uses a grouped batch sampler (several triples of the same instance per batch) so μ_c is estimable in-batch — exactly the structure Sketchy’s ≥5 sketches/photo provides (paper.md §3.3). Also in this notebook: per-instance κ̂_sketch (Banerjee estimator, real sketches only) and the synthetic-vs-real cap diagnostic (paper.md §3.4/§3.6).

Sanity: differentiable log-Bessel vs. scipy

log C_d(κ) needs log I_ν(κ) with ν = d/2 − 1 (255 for d=512). We use the uniform asymptotic expansion (A&S 9.7.7) — differentiable in κ; validate against scipy.special.ive at fp64.

import torch
from scipy import special
from lib import sphere

nu = model.embed_dim / 2 - 1
kap = torch.tensor([1.0, 10.0, 50.0, 200.0, 1000.0], dtype=torch.float64)
ours = sphere.log_bessel_iv(nu, kap)
ref = torch.log(torch.as_tensor(special.ive(nu, kap.numpy()))) + kap   # ive = iv * exp(-k)
for k, o, r in zip(kap, ours, ref):
    err = f"rel.err={abs(o - r) / abs(r):.2e}" if torch.isfinite(r) else "scipy ive underflows to 0 here"
    print(f"kappa={k:7.1f}   ours={o:14.4f}   scipy={r:14.4f}   {err}")

Diagnostics: per-instance κ̂ and the synthetic-vs-real cap

Rules from paper.md §3.4: κ_sketch is estimated from real sketches only; every sample carries a source flag. The cap diagnostic reports κ̂ per source and the angle between the synthetic and real sketch cap means — if synthetic κ̂ ≫ real κ̂ or the angle is large, synthetic sketches are geometrically off-manifold (tighter, differently-centered cap) and must stay augmentation-only. On the mock dataset the “real” sketches are the jittered style and “synthetic” the clean style, so the mechanism is demonstrable end-to-end.

import matplotlib.pyplot as plt
from lib.train import cap_diagnostic, per_instance_kappa_sketch
from lib.run_utils import save_json

kappas = per_instance_kappa_sketch(emb, min_sketches=2, real_only=True)
if kappas:
    fig, ax = plt.subplots(figsize=(6, 3.5))
    ax.hist(kappas, bins=20)
    ax.set(xlabel="per-instance κ̂ (real sketches only)", ylabel="count",
           title="per-instance sketch concentration (Sketchy's ≥5 sketches/photo)")
    fig.tight_layout(); fig.savefig(RUN_DIR / "kappa_per_instance.png", dpi=120); plt.show()
else:
    print("Not enough real multi-sketch instances in the eval subset for per-instance κ̂.")

diag = cap_diagnostic(emb)
print("cap diagnostic:", diag)
if diag.get("synthetic") and diag.get("real"):
    infl = diag["synthetic"]["kappa_hat"] / diag["real"]["kappa_hat"]
    print(f"\nκ inflation (synthetic/real): {infl:.2f}x    cap-mean angle: {diag['cap_angle_deg']:.2f}°")
    print("If inflation >> 1 or the angle is large: synthetic sketches form a tighter,")
    print("differently-centered cap -> augmentation only, never for κ estimation (paper.md §3.4).")
save_json(diag, RUN_DIR / "cap_diagnostic.json")
